# ORBIT - Project Intelligence Operating System

**Intelligent project analysis, critical path optimization, resource planning, and risk mitigation.**

## Features

- **Task Analysis** - Effort validation and slack calculation
- **Critical Path** - Bottleneck identification and compression
- **Resource Planning** - Skill matching and workload balancing
- **Risk Assessment** - Mitigation strategy generation
- **Gantt Charts** - Automated scheduling visualization

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Next.js 14
