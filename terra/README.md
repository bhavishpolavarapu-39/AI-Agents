# TERRA - Climate Intelligence Operating System

**Carbon auditing, climate risk assessment, net-zero pathway design, and sustainability reporting.**

## Features

- **Emissions Inventory** - Scope 1/2/3 quantification
- **Climate Risk** - Physical and transition risk assessment
- **Net-Zero Pathways** - Science-based target setting
- **Regulatory Compliance** - ESG framework alignment
- **Sustainability Reporting** - GRI and TCFD disclosure

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Next.js 14
