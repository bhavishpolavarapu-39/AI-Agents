# LUMINA - Scientific Research Operating System

**AI-powered research planning, hypothesis generation, literature synthesis, and experiment design.**

## Quick Start

```bash
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## Features

- **Hypothesis Generation** - Automated hypothesis formulation from research questions
- **Literature Synthesis** - Semantic search and paper correlation
- **Experiment Design** - Statistical power analysis and methodology planning
- **Knowledge Graphs** - Visual relationship mapping of research concepts
- **Publication Drafting** - NIH proposal-style document generation

## Tech Stack

FastAPI + LangGraph + Claude + Qdrant + PostgreSQL + Next.js 14
