# ATHENA - Adaptive Learning Operating System

**Personalized learning pathways, assessment-based progression, and real-time curriculum adaptation.**

## Quick Start

```bash
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## Features

- **Student Profiling** - Learning style detection and aptitude assessment
- **Adaptive Curriculum** - Dynamic path adjustment based on performance
- **Scaffolded Difficulty** - Progressive challenge scaling
- **Spaced Repetition** - Optimal review scheduling
- **Progress Analytics** - Mastery tracking and skill gaps

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Redis + Next.js 14
