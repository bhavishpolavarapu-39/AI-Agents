# COMPASS - Experience Intelligence Operating System

**User journey mapping, pain point analysis, UX optimization, and satisfaction tracking.**

## Features

- **Journey Mapping** - End-to-end experience visualization
- **Pain Point Analysis** - Friction identification and root cause
- **UX Optimization** - Design improvement recommendations
- **Accessibility Audit** - WCAG compliance checking
- **Satisfaction Metrics** - CSAT, NPS, effort score tracking

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Next.js 14
