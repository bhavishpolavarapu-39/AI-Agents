# LEXIS - Legal Intelligence Operating System

**Intelligent legal analysis, contract review, risk assessment, and regulatory compliance checking.**

## Quick Start

```bash
cd backend && python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## Features

- **Contract Analysis** - Automated clause extraction and risk identification
- **Precedent Search** - Case law and statute correlation
- **Compliance Checking** - Multi-jurisdiction regulatory validation
- **Risk Quantification** - Liability exposure calculation
- **Legal Drafting** - Response and remediation document generation

## Tech Stack

FastAPI + LangGraph + Claude + Qdrant + PostgreSQL + Next.js 14
