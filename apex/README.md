# APEX - Portfolio Intelligence Operating System

**Portfolio analysis, market intelligence, rebalancing strategy, and tax optimization.**

## Features

- **Portfolio Analysis** - Allocation vs target assessment
- **Market Intelligence** - Economic outlook and sector analysis
- **Rebalancing** - Tax-efficient buy/sell recommendations
- **Risk Analysis** - Correlation and drawdown assessment
- **Performance Tracking** - Sharpe ratio and attribution

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Next.js 14
