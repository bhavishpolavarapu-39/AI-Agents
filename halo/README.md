# HALO - Home Intelligence Operating System

**Smart home automation, energy optimization, security monitoring, and device orchestration.**

## Features

- **Device Orchestration** - Unified multi-brand control
- **Automation Rules** - Occupancy and context-aware triggers
- **Energy Optimization** - Usage pattern analysis and cost reduction
- **Security Monitoring** - 24/7 threat detection and alerts
- **Predictive Maintenance** - Equipment health tracking

## Tech Stack

FastAPI + LangGraph + Claude + PostgreSQL + Redis + Next.js 14
