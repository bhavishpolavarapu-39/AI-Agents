# CANVAS - Marketing Intelligence Operating System

**Campaign planning, audience segmentation, creative concept generation, and content optimization.**

## Features

- **Audience Segmentation** - Psychographic and behavioral targeting
- **Creative Concepts** - Multi-variant concept generation
- **Channel Strategy** - Media mix optimization
- **Content Calendar** - 30-day automated planning
- **Performance Prediction** - Expected ROI calculation

## Tech Stack

FastAPI + LangGraph + Claude + Qdrant + PostgreSQL + Next.js 14
